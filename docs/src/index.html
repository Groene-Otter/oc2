    1. @import "@groene-otter/oc2/main.css";
    2. @import "npm:@groene-otter/oc2/main.css";

Getting started

Brand color

All you need to do is set a brand color in oklch format. OC2 is based on a very light brand color, such as 90% lightness in oklch.

Border radius

If you want a straight corner look, simply set the default border radius to 0. Check the example below

Examples

These are the default values in Open Components.

html {
  --brand: oklch(90% 0.186 148);
  --default-radius: var(--radius-2);
}

Here's an example for a pink theme that's a bit darker and with straight corners. 

html {
  --brand: oklch(80% 0.226 330);
  --default-radius: 0;
}
